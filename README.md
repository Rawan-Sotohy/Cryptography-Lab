# Cryptography-Lab

A MATLAB-based cryptography project implementing classical and modern encryption algorithms.

## Implemented Ciphers

* Caesar Cipher
* Vigenère Cipher
* Playfair Cipher
* Affine Cipher
* Hill Cipher
* Rail Fence Cipher
* Columnar Cipher
* Double Transposition Cipher
* RC4
* A5/1
* RSA
* One-Time Pad (OTP)

## Features

* Encryption and Decryption
* Interactive command-line interface
* Modular project structure
* Multiple cryptographic techniques in one toolkit

## Project Structure

```text
Cryptography-Lab/
│
├── main.m
├── encrypt.m
├── decrypt.m
│
├── classical/
├── stream/
├── modern/
```

## How to Run

1. Open the project folder in MATLAB.
2. Run:

```matlab
main
```

3. Choose:

* Cipher type
* Encryption or decryption
* Text
* Key

## Example

```text
Choose Cipher: 1
Choose Mode: 1
Enter Text: hello
Enter Key: 3

Encrypted Text: KHOOR
```
